Projet : Application de Banque en Ligne

Technologies utilisées : Python (Flask/Django), HTML/CSS, SQLite

Ce projet est une application de gestion bancaire permettant aux utilisateurs de :

  • Créer un compte client
  • Consulter leur solde
  • Effectuer des virements
  • Voir l’historique de leurs transactions

Le backend a été développé avec Python, en utilisant le framework Flask (ou Django, selon ton projet). Les données sont stockées dans une base SQLite, et les sessions utilisateurs sont sécurisées avec des tokens.
